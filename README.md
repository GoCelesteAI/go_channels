# Go Tutorial - Lesson 25: Channels

This repository contains the source code for **Go Tutorial for Beginners #25 - Channels**.

## Files

- `channel_basics.go` - Creating channels, sending and receiving
- `channel_buffered.go` - Buffered vs unbuffered channels
- `channel_close.go` - Closing channels and the comma-ok idiom
- `channel_range.go` - Range over channels and pipeline pattern

## Running the Examples

```bash
go run channel_basics.go
go run channel_buffered.go
go run channel_close.go
go run channel_range.go
```
